# Github Collaborator 추가

- Github `Repository` 이동
- `Settings` 이동
- `Collaborators and teams` 선택
- `Add People` 선택
- 사용자 이메일 또는 닉네임으로 초대
- 초대받은 사용자가 초대버튼 누르면 완료
